LED Temp Humidity Lamp

An innovative LED lamp that changes color based on real-time temperature and humidity readings using a Raspberry Pi and DHT11 sensor. The lamp visually indicates whether the environmental conditions fall within the desired range, offering an intuitive way to monitor and maintain optimal conditions

Language/Tools Used: Python, Raspberry Pi, DHT11 Sensor, RGB LED, LCD Display

Integral Approximation App

A web-based tool that helps users estimate the value of definite integrals for polynomials with degree up to 2. It allows users to input polynomial coefficients, define integration bounds, and visualize the result with interactive graphs, including a shaded area representing the approximated integral, while displaying the integral expression.

Language/Tools Used: React(Js), HTML, CSS, MathJax, Ploty.js
